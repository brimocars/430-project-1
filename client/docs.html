/state POST

Query Parameters

state - The state to update the senators for

Body Parameters

firstName (string) - A senator's first name
lastName (string)- A senator's last name
name (string) (optional) - A senator's full name, including their title and party-state code. If not provided, one will be generated based on the other parameters. If provided, it will be used to search for a senator to update rather than create a new one.
party (string) - A senator's party
state (string) - A senator's state - must match the state in the query parameter
congress_numbers (number[]) - The 3 congress numbers that a senator's term is for
any (any) - Any additional data about this senator

Returns

A 204 No Content

Example

Request: /state [{ "person": { "firstname": "1", "lastname": "2" }, "state": "MD", "party": "some very real party", "congress_numbers": [1,2,3]}, { "person": { "firstname": "1", "lastname": "2" }, "state": "Md", "party": "some very real party", "congress_numbers": [1,2,3]}]
204
